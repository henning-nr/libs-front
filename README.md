# libs-front
Projeto apresentando as principais libs de desenvolvimento de aplicações frontend.

## Como executar

```bash
node server.js
```

Abra `http://localhost:3000` no navegador.

## Rotas disponíveis

A página inicial (`/`) lista as rotas das demonstrações:

- `/vanilla`
- `/jquery`
- `/react`
- `/angularjs`
- `/w3js`

Cada rota serve um HTML próprio da biblioteca, carregando somente o script e CDN necessários para aquela implementação.

## Diferenças de abordagem (resumo)

- **Vanilla JS**: manipulação direta do DOM com APIs nativas.
- **jQuery**: simplifica seleção de elementos e eventos.
- **React**: muda a interface através de estado do componente.
- **AngularJS**: usa data binding e diretivas no template.
- **W3.JS**: oferece utilitários simples para manipulação de classes/elementos.
